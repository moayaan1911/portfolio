# 🌐 Moayaan's 3D Animated Decentralized Portfolio

Welcome to the 3D Animated Decentralized Portfolio of Mohammad Ayaan Siddiqui, a self-taught Full Stack Blockchain Developer and Decentralization MAXI. This portfolio showcases my skills, projects, and social links in an interactive and engaging manner.

## 🚀 Tech Stack

This project is built using the following technologies:

- **Next.js**: A React framework for server-side rendering and static site generation.
- **React**: A JavaScript library for building user interfaces.
- **Three.js**: A JavaScript library for creating 3D graphics in the browser.
- **Framer Motion**: A library for animations and gestures in React.
- **Framer Motion 3D**: Extends Framer Motion to work with Three.js.
- **Jotai**: A state management library for React.
- **Tailwind CSS**: A utility-first CSS framework for styling.
- **React Icons**: A collection of popular icons for React.
- **GSAP**: A powerful animation library.
- **Leva**: A GUI for tweaking parameters in React.

## ✨ Features

- **3D Animations**: Interactive 3D elements and animations using Three.js and Framer Motion 3D.
- **Decentralized Hosting**: Deployed on IPFS and Fleek for a decentralized web experience.
- **Responsive Design**: Optimized for both desktop and mobile devices.
- **Interactive UI**: Engaging user interface with smooth animations and transitions.
- **Social Links**: Easy access to my social profiles and contact information.

## 🛠️ How to Run Locally

To run this project locally, follow these steps:

1. **Clone the repository**:

   ```bash
   git clone https://github.com/moayaan1911/portfolio.git
   cd your-repo-name
   ```

2. **Install dependencies**:

   ```bash
   npm install
   ```

3. **Run the development server**:

   ```bash
   npm run dev
   ```

4. **Open your browser** and navigate to `http://localhost:3000`.
